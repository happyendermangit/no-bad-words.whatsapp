## How to use?
1. clone the repo:
```sh
$ git clone https://github.com/happyendermangit/no-bad-words.whatsapp.git
```

2. Install modules
```sh
$ npm install whatsapp-web.js qrcode-terminal
```
3. add the bad words list in [./words.js](./words.js)
4. Run the bot
```sh
node index.js
```
5. Login with qr code
6. enjoy
